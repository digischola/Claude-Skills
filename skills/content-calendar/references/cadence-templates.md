# Cadence Templates (Weekly Rhythm)

Source: channel-playbook.md (LOCKED Phase 1 plan). This file codifies the per-week slot grid the calendar generator fills.

## Phase 1 rhythm (weeks 1-6: LinkedIn reactivation)

Primary channel: **LinkedIn** (80% of effort)
Secondary (repurpose only): Instagram + X + Facebook
Dark: WhatsApp (seed bio links, no posts yet)

### Weekly slot grid (Phase 1)

| Day | Channel | Format | Default pillar | Notes |
|---|---|---|---|---|
| **Monday** | LinkedIn | text-post | Pillar 1 (LP Craft) | Kick the week with a visible-result case. Data or Story hook preferred. |
| **Tuesday** | X | thread | (repurpose Monday LI) | Repurpose Monday into a 5-7 tweet thread. Post before noon local. |
| **Tuesday** | X | single tweet | (fresh observation) | 1 short take during the day. |
| **Wednesday** | LinkedIn | text-post | Pillar 2 (Solo Freelance Ops) | Honest operator take. Personal-stake or Timeline hook preferred. |
| **Thursday** | LinkedIn | carousel | flex (any pillar, format-fit priority) | Highest-engagement format (7%). Use for structured frameworks or before/after. |
| **Thursday** | X | single tweet | (fresh) | Light. |
| **Friday** | LinkedIn | text-post | Pillar 3 (Small-Budget Paid Media) | Campaign diagnosis or budget-tier take. Counterintuitive or Data hook. |
| **Friday** | X | thread | (repurpose Friday LI) | Repurpose. |
| **Saturday** | Instagram | Reel OR carousel | flex (repurpose best-of-week) | Turn the week's strongest LI post into visual content. |
| **Saturday** | X | single tweet | (fresh light take) | Weekend tone. |
| **Sunday** | — | engagement-outbound day | — | No original posts. Use for engagement-outbound (future skill) + next-week calendar planning. |

**Totals per week:**
- LinkedIn: 4 posts (Mon / Wed / Thu / Fri)
- Instagram: 1 post (Sat) — can scale to 2-3 as IG audience grows
- X: 5-7 tweets (2-3 threads repurposed + 3-4 single tweets)
- Facebook: 1 LI repurpose (schedule Mon or Thu, no original content)
- WhatsApp: dark Phase 1

## Phase 3 rhythm (weeks 13+: optimize)

Driven by performance-review data. Cadence adapts based on what's working. No static template.

## Pillar rotation rules

Monday → Wednesday → Friday default:
- Mon = Pillar 1 (Landing-Page Conversion Craft)
- Wed = Pillar 2 (Solo Freelance Ops)
- Fri = Pillar 3 (Small-Budget Paid Media)
- Thu + Sat = flex (highest-value entry regardless of pillar)

If a week has 2+ raw entries in one pillar and 0 in another, the rotation may bend: Thursday's flex slot can be used to rebalance (e.g., if Friday has no Pillar 3 entry, skip Friday LI and use Thursday's carousel for whatever's available).

## Repurpose mapping (1 idea → multiple slots)

A single idea-bank entry can legitimately fill multiple slots in a week when it's a strong one:

| Source slot | Auto-repurpose targets |
|---|---|
| Monday LinkedIn text-post | Tuesday X thread + Saturday IG Reel or carousel |
| Wednesday LinkedIn text-post | Wednesday X single tweet (quote from post) |
| Friday LinkedIn text-post | Friday X thread + Saturday carousel |
| Thursday LinkedIn carousel | Thursday X single tweet (slide 1 as visual) |

This is how a sparse idea-bank (5-6 entries) can still fill a full week.

## Format variety rules

Never ship 3 identical formats in a row.

- If Mon LinkedIn = text-post and Wed LinkedIn = text-post, Thursday must be carousel or video.
- If 3 LinkedIn posts in a week are all text, the 4th must be carousel or native video.

## AI-theme cadence

Approximately 1 in 3 LinkedIn posts should explicitly weave an AI-workflow angle (Claude, Perplexity, Remotion, Lovable, Wabo). Over a 4-post week, that means 1-2 AI-themed posts, not 3+.

Track this week-by-week in the calendar. If a week runs 3+ AI-heavy posts, flag as over-indexing — rebalance the next week toward pure case-study or operator-insight content.

## Time-of-day defaults (IST)

Based on channel-playbook.md + platform-specs.md:

- **LinkedIn:** Tue-Thu 8-10 AM IST (US evening browsing, AU + SG morning)
- **X:** Tue-Thu 12-6 PM local (use IST but note engagement is global)
- **Instagram:** weekday evenings or Saturday morning
- **Facebook:** match LinkedIn time when repurposing

Scheduler-publisher (future skill) reads this from the calendar file.
