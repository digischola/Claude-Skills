# Hook Library (35 Proven Patterns, Adapted for Digischola)

Source: Perplexity research 2026-04-18. Each pattern tagged with channel fit + engagement signal + when-to-use. All examples rewritten in Mayank's voice with real Digischola data (Thrive, ISKM, Samir's) where possible.

## Usage rules

1. Pick 3 candidate hooks per post. Never lock one without user preview.
2. Match hook category to entry type — see `post-writer/references/entry-to-hook-map.md` if it exists, otherwise use the "Best for" field below.
3. Never reuse the same hook template within 2 weeks for the same pillar.
4. Hook must pass validator (em dash ban, hype word ban, 100-150 char target for LinkedIn).

## Anti-patterns (never ship)

- "Let me tell you about..." / "Here is a thought..." / "Ever wondered..."
- Hooks that promise "one weird trick" or "the secret to..."
- Generic questions with no stake ("What are your thoughts on AI?")
- Hooks longer than the LinkedIn truncation fold (~210 chars)
- Any hook containing an em dash
